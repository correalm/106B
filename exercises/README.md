**Iniciando**

```bash
git clone https://github.com/cpputest/cpputest.git
cd cpputest
./autogen.sh
./configure
make
sudo make install
```
---

Rodando os testes:

`make && ./test_runner`

Depois da execução:

`make clean`

---
